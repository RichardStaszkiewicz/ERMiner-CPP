\doxysubsection{Detailed Description}
Implements the \doxylink{classERMiner}{ERMiner} algorithm for efficient rule extraction. 

The \doxylink{classERMiner}{ERMiner} class uses equivalence class-\/based search to mine association rules from sequence databases, with customizable support and confidence thresholds.
