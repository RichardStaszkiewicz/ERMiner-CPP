\chapter{ERMiner\+: A C++ Implementation for Rule Mining}
\hypertarget{index}{}\label{index}\index{ERMiner: A C++ Implementation for Rule Mining@{ERMiner: A C++ Implementation for Rule Mining}}

\doxylink{classERMiner}{ERMiner} is a C++ implementation of a frequent sequential rule mining algorithm. It is designed to efficiently discover frequent rules in sequential databases. The project supports operations such as equivalence class generation, rule validation, and exporting results to a CSV file.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Features}{Features}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item Frequent \doxylink{classRule}{Rule} Mining\+: Identify rules that meet user-\/defined minimum support (minsup) and confidence (minconf) thresholds.
\item Equivalence Classes\+:
\item Left Equivalence Class (\+\_\+find\+Left\+Equivalence\+Classes)\+: Groups rules by their antecedents.
\item Right Equivalence Class (\+\_\+find\+Right\+Equivalence\+Classes)\+: Groups rules by their consequents.
\item Recursive Search\+:
\item \+\_\+left\+Search\+: Expands rules by appending items to the consequent.
\item \+\_\+right\+Search\+: Expands rules by appending items to the antecedent.
\item CSV Export\+: Save the discovered rules to a CSV file with their respective support and confidence values.
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysection{\texorpdfstring{Project Structure}{Project Structure}}\label{index_autotoc_md2}
