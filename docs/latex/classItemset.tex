\doxysubsection{Detailed Description}
Represents a set of items with additional functionality for sequence analysis. 

The \doxylink{classItemset}{Itemset} class is an extension of {\ttfamily std\+::set\texorpdfstring{$<$}{<}int\texorpdfstring{$>$}{>}} with additional methods for managing occurrences, generating string representations, and computing relationships with sequences.
