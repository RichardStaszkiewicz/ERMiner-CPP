\doxysubsection{Detailed Description}
Represents an association rule with methods for support, confidence, and validation. 

The \doxylink{classRule}{Rule} class manages an antecedent and a consequent (both {\ttfamily \doxylink{classItemset}{Itemset}} objects), with functions to compute the rule\textquotesingle{}s validity in a sequence database.
