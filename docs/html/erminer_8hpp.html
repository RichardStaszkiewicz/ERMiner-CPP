Detailed Description

Declaration of the ERMiner class for extracting association rules from sequence databases.

The ERMiner algorithm is an efficient method for mining association rules, based on the approach described in ERMiner: sequential rule mining using equivalence classes by Fournier-Viger et. al. It identifies frequent and valid rules by leveraging equivalence class search strategies.
